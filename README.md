# SD1_Project_Installer
This repository contains the final project for the Software Development 1 course. The project focuses on creating a functional Windows application along with a customized installation package.

# Important Note for Evaluators
The installer in this project is specifically configured to meet the course requirements regarding installation paths and organization naming.

# Project Specifications
The Windows Installer (.msi) has been built with the following parameters:

Manufacturer/Organization: VVK

Product Name: Mesut_Gok

Default Installation Path: C:\Program Files (x86)\VVK\Mesut_Gok\

Main Executable: SD1_GUI.exe

# Repository Structure
SD1_GUI/: Contains the full C# source code for the application.

SD1_Setup/: Contains the Visual Studio Setup Project files and the generated .msi installer.

SD1_GUI.sln: The Visual Studio Solution file to open the entire project.

SD1_GUI.zip: A compressed package of the entire project for easy downloading.

# Installation Instructions
Navigate to the SD1_Setup/Debug/ folder.

Download and run SD1_Setup.msi.

Follow the setup wizard. The application will be installed under the VVK directory in Program Files.
